📚 BookSpace – Online Library Management System (OLMS)

BookSpace is a modern, browser-based Online Library Management System (OLMS) designed to simplify and automate library operations. It provides an intuitive interface for both members (students) and administrators (librarians) to manage books, borrowing, returns, and fines efficiently.

👤 Member Features
 1. User registration and login system
 2. Browse and search books by title, author, or category
 3. Borrow and return books
 4. View borrowed book history
 5. Automatic fine calculation for late returns
 6. Profile management (edit personal details)

🛠️ Admin Features
1. Dashboard with system analytics
2. Add, update, and delete books
3. Manage book inventory and stock
4. View all members and their details
5. Monitor borrowed books and force return
6. Clear member fines (waiver option)

💻 Technologies Used
1. HTML5 – Structure of the application
2. CSS3 – Styling and responsive UI (Dark theme UI)
3. JavaScript (Vanilla JS) – Core functionality and logic
4. LocalStorage – Data storage (no backend required)

⚙️ How to Run the Project
 1. Download or clone the repository
 2. Open the project folder
 3. Double-click index.html or open it in a browser
 4. Start using the application
